## Directory Structure Overview

- **Root Level**: Ruby on Rails application configuration files (Gemfile, Dockerfile, etc.)
- **.git/**: Git version control system files and objects
- **app/**: Main application code (assets, controllers, models, views, etc.)
- **bin/**: Executable scripts
- **config/**: Application configuration files and initializers
- **db/**: Database related files
- **lib/**: Library code and custom tasks
- **log/**: Application log files (currently empty except for .keep)
- **public/**: Static files served directly by web server (including compiled assets)
- **storage/**: File storage location (currently empty except for .keep)
- **test/**: Test files and fixtures
- **tmp/**: Temporary files and caches (including Bootsnap compilation cache)
- **vendor/**: Third-party code and dependencies
